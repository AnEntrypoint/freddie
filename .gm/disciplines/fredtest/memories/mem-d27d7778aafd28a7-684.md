Q: Write a simple node.js express hello world server to c:\dev\fredtest\server.js
A: Sure, I can help with that. Here's a simple Node.js Express "Hello World" server that you can save to `c:\dev\fredtest\server.js`:

```javascript
const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('Hello World!');
});

app.listen(port, () => {
  console.log(`Server is running at http://localhost:${port}`);
});
```

To run this server, you'll need to have Node.js installed on your machine. You can download it from [Node.js official website](https://nodejs.org/).
